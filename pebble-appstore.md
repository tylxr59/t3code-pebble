## Name

T3 Code for Pebble

## Description

Keep your T3 Code agents moving from your Pebble Time 2.

T3 Code for Pebble is a tiny remote control for a reachable T3 Code workspace. Browse active projects and threads, read recent model responses, check whether an agent is still working, and dictate a new prompt without opening a laptop or phone.

Features:

- Browse active T3 Code projects and non-archived threads
- Read recent user and model messages
- Check live thread status from your wrist
- Start a new thread
- Dictate and send prompts to an existing thread
- Authenticate through the companion app with a one-time pairing code

Requires a Pebble Time 2, a paired phone, and a reachable T3 Code instance.
